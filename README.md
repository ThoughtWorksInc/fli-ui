# FLI UI

## Getting Started
```
$ npm install
```

## Running the application
```
$ npm run dev
```

## Running tests
```
$ npm test unit
$ npm test interaction
$ npm test contract
```

This will make the application accessible at http://localhost:8080

The production site is available at: http://fli-ui-artefacts.s3-website-us-west-1.amazonaws.com/
